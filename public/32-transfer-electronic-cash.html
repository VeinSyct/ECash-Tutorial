To change currency you must call the function below

SCRIPT 

    function transferAmount(d) {
        _uz.api.contentWindow.postMessage({
            action: "ecash-api-transfer-amount",
            currency: document.querySelector(".beneficiary-currency").value,
            range: document.querySelector(".beneficiary-amount").value
        }, _uz.api.src);
    };

Add html elements to show the acquired beneficiary information in your html page

HTML

Add QR functionality to share information via QR code, you need to install a QR code generator library.

In the repository folder look for the file qr.creator.js and add the line below in your html page.

HTML 

    <script type="module">
        import QrCreator from "./assets/js/qr.creator.js";
        // Your codes here...
        // ...
        // Your codes here...
    </script>

Add html element to show the QR code in your html page

HTML
